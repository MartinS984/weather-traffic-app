# 🌦️🚗 Weather & Traffic IoT Microservices

> 🔗 **Companion Repository:** This repository contains the **Node.js application source code** for the telemetry producer and consumer. The fully automated GitOps infrastructure (Kafka, Prometheus, Grafana, ArgoCD) used to deploy and run these microservices can be found here: [**MartinS984/weather-traffic-gitops**](https://github.com/MartinS984/weather-traffic-gitops)

This project consists of two containerized microservices that generate IoT telemetry and bridge it to Prometheus for real-time visualization.
